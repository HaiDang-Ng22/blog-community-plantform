Zynk RESTful API

Tài liệu tích hợp hệ thống, hướng dẫn xác thực JWT và danh sách endpoints mở rộng.

Môi trường & Base URL

Mọi kết nối API chính thức đều thông qua cổng giao thức an toàn HTTPS với cấu trúc đường dẫn cơ sở:

Production: https://zynk.app/api/
Development: http://localhost:7000/api/

Xác thực (Authentication)

Các request cần xác thực yêu cầu truyền JWT token trong Header:

Authorization: Bearer <jwt_token>

Lấy Token (Login):

POST /api/auth/login
// Payload:
{
  "email": "dev@zynk.app",
  "password": "secure_pass"
}

// Response (200 OK):
{
  "token": "eyJhbGciOiJIUzI1NiIs..."
}
