Zynk Developer API

Tích hợp sức mạnh của Zynk vào ứng dụng của bạn với REST API đơn giản và mạnh mẽ.

RESTKiến trúc
JWTXác thực
JSONĐịnh dạng

Base URL

/api/

Tất cả các endpoints đều yêu cầu header Authorization: Bearer <token> (trừ đăng nhập/đăng ký).

Xác thực

Endpoint để lấy JWT token:

POST /api/auth/login

{
"email": "user@example.com",
"password": "your_password"
}

// Response
{
"token": "eyJhbGciOiJIUzI1NiIs..."
}

Endpoints chính

  • GET/api/marketplace/products – Lấy danh sách sản phẩm
  • GET/api/marketplace/categories – Lấy danh mục
  • GET/api/posts – Lấy danh sách bài viết
  • POST/api/posts – Tạo bài viết mới
  • GET/api/users/{id}/profile – Lấy hồ sơ người dùng

Yêu cầu API Key

API hiện đang trong giai đoạn beta. Liên hệ chúng tôi để được cấp quyền truy cập sớm.
